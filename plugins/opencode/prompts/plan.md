## Task

Create a detailed implementation plan for the following:

{{prompt}}

## Context

Working directory: {{cwd}}

## Output Format

### 1. Overview
One paragraph summary of the approach.

### 2. Files to Create/Modify
List each file with:
- **Path**: full file path
- **Action**: create / modify / delete
- **Changes**: what to add or change

### 3. Dependencies
Any new packages, APIs, or services needed.

### 4. Implementation Steps
Numbered step-by-step plan. Each step should be independently completable.

### 5. Key Decisions
Architecture choices and their tradeoffs.

### 6. Risks & Mitigations
What could go wrong and how to handle it.

### 7. Testing Strategy
How to verify the implementation works.

Keep under 1000 words. Be specific about file paths and code patterns.
